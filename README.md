

## Project Overview

This project demonstrates advanced computer vision and 3D data processing techniques through two main experiments: feature matching for stereo correspondence and point cloud registration. The first module implements camera calibration, feature extraction, and depth estimation using stereo image pairs, producing visual outputs and depth maps for both training and test sets. The second module focuses on 3D point cloud alignment, employing nearest neighbor search and pose estimation to register and visualize real-world 3D datasets. The project showcases proficiency in Python, OpenCV, NumPy, and data visualization, and is structured for reproducible experimentation and portfolio presentation.
